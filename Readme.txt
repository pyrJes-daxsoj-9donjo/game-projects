Змейка на Pygame

Описание

Этот проект представляет собой классическую игру "Змейка", реализованную с использованием библиотеки Pygame. Игрок управляет змейкой, собирает фрукты и избегает столкновений с границами и врагами. В игре есть система очков и жизней.

Функционал

Управление змейкой с помощью стрелок на клавиатуре.

Различные виды фруктов с разным количеством очков.

Случайно появляющийся враг, который движется в случайном направлении.

Границы, ограничивающие игровое поле.

Отображение количества очков и оставшихся жизней.

Возможность перезапуска игры после поражения.

Установка и запуск

Убедитесь, что у вас установлен Python (версии 3.x).

Установите библиотеку Pygame, если она ещё не установлена:

pip install pygame

Запустите игру командой:

python pygame.py

Управление

Стрелка влево (←) – движение влево

Стрелка вправо (→) – движение вправо

Стрелка вверх (↑) – движение вверх

Стрелка вниз (↓) – движение вниз

Q – выход из игры после поражения

R – перезапуск игры после поражения

Как играть

Управляйте змейкой, собирая фрукты, чтобы увеличивать счёт и длину змейки.

Избегайте столкновений с границами, врагами и своим хвостом.

При столкновении с врагом или границами теряется одна жизнь. Когда жизни заканчиваются, игра завершается.
